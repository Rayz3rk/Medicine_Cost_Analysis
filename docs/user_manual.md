# 药品成本分析系统 操作手册

## 1. 系统简介
本系统通过集成大型语言模型和多智能体协同架构，实现药品生产、运输、储存等全链路的成本精算与分析，并自动生成结构化的分析报告。

## 2. 登录与主界面
- **登录**：访问系统首页，使用管理员分配的账号密码进行登录。
- **主界面**：包含左侧导航栏、顶部控制区（支持中英文切换、暗黑模式切换）以及主体内容区。

## 3. 药品管理
1. 在左侧菜单点击“药品列表”。
2. **新增**：点击右上角“新增药品”按钮，填写药品名称、规格、适应症等基本信息。
3. **编辑/删除**：在表格的操作列中，可以对已有药品信息进行修改或移除。所有操作将实时同步至数据库。

## 4. 成本分析
1. 在左侧菜单点击“成本分析”。
2. **选择药品**：在下拉框中选择需要测算的药品。
3. **选择单位**：支持元/100片、元/剂、元/盒三种单位。
4. **填写成本**：
   - 生产成本（必填原料成本）
   - 运输成本
   - 储存成本
   - **其他自定义成本**：点击“添加成本项”，输入成本名称和金额，系统会自动将其计入总成本。
5. **开始分析**：点击提交按钮，系统将在后台调度“成本会计”、“定价策略师”、“供应链经理”三个智能体进行联合分析。
6. **查看结果**：分析完成后，右侧面板将显示饼图、总成本评估及各专家的具体建议。

## 5. 报告导出
在“成本分析”结果页面的右上角，点击：
- **导出 PDF**：生成不可篡改的正式版 PDF 报告。
- **导出 Word**：生成可二次编辑的 DOCX 报告。

## 6. 系统安全与合规
- 您的数据将经过 AES 加密处理后存储。
- 操作日志可在系统后台审计查询。
